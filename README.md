# Asterisp

![logo](logo/logo_dark.svg)

A simple interpreted lisp.

## Usage

**REPL**

Start a REPL by running the executable

**Execute file**

Pass a file to execute as the first argument to the executable

(examples are available in the `examples` directory)

## Build

``` sh
dub build
```
